Инструкция по установке и запуску.

Метод 1: Запуск из архива (рекомендуется)

Шаг 1: Загрузка файлов
1. Загрузите архив polymer_viscosity.zip
2. Сохраните архив в удобную для вас папку на компьютере

Шаг 2: Распаковка архива
1. Щелкните правой кнопкой мыши по файлу polymer_viscosity.zip
2. Выберите "Извлечь все..." (Windows) или "Распаковать архив" (macOS/Linux)
3. Укажите папку для распаковки (например, C:\Programs\PolymerCalc или /home/user/PolymerCalc)
4. Нажмите "Извлечь" или "OK"

Шаг 3: Проверка содержимого
index.html - главный файл приложения
styles.css - файл стилей 
script.js - файл скриптов 

Шаг 4: Запуск приложения
1. Найдите файл index.html в папке с распакованными файлами
2. Дважды щелкните по файлу index.html
3. Приложение автоматически откроется в вашем браузере по умолчанию
